Vehicle Detection & Tracking using YOLOv8
📌 Overview

This project focuses on real-time vehicle detection and tracking using state-of-the-art computer vision techniques. It leverages YOLOv8 for detection, ByteTrack for tracking, and Supervision for visualization and pipeline management.

The system is capable of identifying and tracking multiple vehicles in video streams efficiently.

🎯 Features
🚘 Real-time vehicle detection
🔄 Multi-object tracking using ByteTrack
📊 Bounding boxes with unique IDs
🎥 Works on video files or live streams
⚡ High accuracy and performance
🛠️ Tech Stack
Detection Model: YOLOv8
Tracking Algorithm: ByteTrack
Visualization & Utilities: Supervision
Language: Python
Environment: Google Colab / Local setup
